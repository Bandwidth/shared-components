```
const theme = require('../../theme').default;
theme.renderDocumentation('Alert');
```

Alerts control their own color based on \`type\`.

```
<Alert type="info">
    Hello, world
</Alert>
```

```
<Alert type="info" textOnly>
    Hello, world
</Alert>
```

```
<Alert type="success">
    Hello, world
</Alert>
```

```
<Alert type="error">
    Hello, world
</Alert>
```
