Horizontal is a simple layout tool that lays items out horizontally left-aligned with spacing. It uses `display: flexbox;` to lay items out.

```js
<Horizontal>
  <Button>First</Button>
  <Button>Second</Button>
  <Button>Third</Button>
</Horizontal>
```

#### Custom spacing

```js
<Horizontal spacing="xl">
  <Button>First</Button>
  <Button>Second</Button>
  <Button>Third</Button>
</Horizontal>
```
