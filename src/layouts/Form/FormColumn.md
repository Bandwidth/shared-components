# FormColumn

Sometimes forms are split into vertical columns. Very rarely, but we still have a component for that.

\`\`\`
<Form>
  <FormColumn>
    <Field component={Foo} />
    <FieldGroup> ... </FieldGroup>
  </FormColumn>
  <FormColumn>
    <Field component={Bar} />
    <FieldGroup> ... </FieldGroup>
  </FormColumn>
</Form>
\`\`\`
