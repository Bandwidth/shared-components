LogoHeader
==========


Props
-----

Prop                  | Type     | Default                   | Required | Description
--------------------- | -------- | ------------------------- | -------- | -----------
children|string|'Bandwidth'|No|

LogoHeader is mostly used for the top nav. It's just like Header, except it renders a logo before the text you supply.
