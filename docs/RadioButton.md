RadioButton
===========


Props
-----
Prop                  | Type     | Default                   | Required | Description
--------------------- | -------- | ------------------------- | -------- | -----------
checked|bool||Yes|
onChange|func||Yes|
label|string||Yes|
content|node|null|No|
name|string||Yes|
value|string||Yes|

Not really meant to be used alone, unless you just want the styling of it. Use RadioGroup to create a group of RadioButtons.
