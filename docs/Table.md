Table
=====


Props
-----

Prop                  | Type     | Default                   | Required | Description
--------------------- | -------- | ------------------------- | -------- | -----------
children|node||Yes|
headers|node||Yes|
loading|bool|false|No|

Renders a table, using the `children` and `headers` props to define the various table parts. Also accepts `loading` to show a loading state.
